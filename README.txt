*******************************

Program lets user purchase movie ticket and concessions. The program asks the user to enter the movie title, movie time, how many tickets the user needs and the amount of concessions user needs. The program also lets user enter an available coupon code to get a discount off of the final price paid and the ability to refund a movie ticket if the movie has not started yet.

*******************************

Compile:

	-ubuntu virtual machine
		-Terminal:
			make
			./MovieProgram
